## Requirements

- Docker Compose : `v2.28.1`
- Make : `v4.3`

## Quick start

1. Run `make run`
   - Wait for services to be healthy
2. Access Django at [http://localhost:8000](http://localhost:8000)
3. Access FastAPI at [http://localhost:8000](http://localhost:8000)
   - User: `guestready`
   - Password: `test123`
